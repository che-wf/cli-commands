# Git

**Fetch:**

```
git fetch
```

**Fetch from remote:**

```
git fetch [remote]
```

**Merge with another branch:**

```
git merge [branch]
```

**Options:**

* Merge with unrelated histories: `--allow-unrelated-histories`

**Set remote:**

```
git remote add [remote name] git@github.com:[account]/[repo].git
```
