# n

Note: use `sudo` on Linux or WSL

**To install:**

```
npm i -g n
```

**To install versions of node:**

```
n latest
```

**Options**:

* `latest`
* `stable` 
* `lts`
* version number

**Remove a version:**

```
n rm 8.9.0
```

or

```
n - 8.9.0
```

**Prune all packages but current:**

```
n prune
```
