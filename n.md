# npm - n

Note: use `sudo` on Linux or WSL

### **To install:**

```text
npm i -g n
```

### **To install versions of node:**

```text
n latest
```

#### **Options**:

* `latest`
* `stable` 
* `lts`
* version number

### **Remove a version:**

```text
n rm 8.9.0
```

or

```text
n - 8.9.0
```

### **Prune all packages but current:**

```text
n prune
```
